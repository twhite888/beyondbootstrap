Font Awesome

Vector graphics that scale nicely

No drawing skills or images required.

  1. Go to http://fontawesome.io/get-started/ and submit your email address
  2. Add the CDN JS link from your email into the <head>
  3. Add the icon link to the html: e.g. <i class="fa fa-home"></i> Refer to:
    • http://fontawesome.io/examples/ for examples of how to use
    • http://fontawesome.io/icons/ for the list of icons
  4. Set the color and spacing of the icon in the stylesheet using .fa or individual icon class names
